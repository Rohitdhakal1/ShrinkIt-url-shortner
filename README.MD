## ğŸ—ï¸ System Architecture

**The Data Flow:**

1.  **Write (Shorten):** User -> React -> API -> MongoDB (Persist) -> Redis (Cache) -> User.
2.  **Read (Redirect):** User -> API -> **Redis (Hit?)** -> Redirect.
    - _If Miss:_ -> MongoDB -> Save to Redis -> Redirect.
